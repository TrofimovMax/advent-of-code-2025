## --- День 3: Лобби ---

Вы спускаетесь по небольшой лестнице, входите в удивительно просторное лобби и быстро проходите проверку безопасности. Однако, когда вы доходите до главных лифтов, обнаруживаете, что над каждым горит красный индикатор: все лифты отключены.

«Извините за это», — говорит Эльфийка, ковыряясь в панели управления неподалёку. — «Похоже, какой-то электрический скачок их выжег. Постараюсь включить их как можно скорее».

Вы объясняете, что вам нужно спуститься глубже под землю.  
«Ну, вы хотя бы можете воспользоваться эскалатором до отдела печати, хотя дальше вы всё равно не попадёте без работающих лифтов. То есть могли бы… если бы эскалатор тоже не был выключен».

«Но не переживайте! Его не спалило; ему просто нужна энергия. Возможно, вы сможете его запитать, пока я работаю над лифтами».

Рядом лежат батареи, которые могут обеспечить аварийное питание для эскалатора. На каждой батарее указано её напряжение — число от 1 до 9. Вы записываете их значения (ваш входной набор данных). Например:

```
987654321111111
811111111111119
234234234234278
818181911112111
```


Батареи организованы в блоки — каждая строка входных данных соответствует одному блоку батарей.  
Внутри каждого блока нужно включить **ровно две** батареи; напряжение блока равно числу, образованному из двух включённых цифр **в порядке их расположения в строке**.

Например, если есть блок `12345` и вы включаете батареи 2 и 4, блок выдаёт напряжение **24**. (Менять порядок батарей нельзя.)

Ваша задача — найти **максимальное возможное напряжение** для каждого блока.  
Для примера выше:

- В `987654321111111` максимальное возможное напряжение — **98** (первые две батареи).
- В `811111111111119` максимальное напряжение дают батареи `8` и `9`, то есть **89**.
- В `234234234234278` можно получить **78**, включив последние две батареи (`7` и `8`).
- В `818181911112111` максимальное напряжение — **92**.

Общее выходное напряжение — сумма максимальных напряжений всех блоков:  
**98 + 89 + 78 + 92 = 357**.

Перед вами множество батарей.  
Найдите максимальное возможное напряжение для каждого блока:  
**каково суммарное выходное напряжение?**

--- Часть вторая ---

Эскалатор не двигается. Эльф объясняет, что, вероятно, системе нужно больше напряжения, чтобы преодолеть статическое трение, и нажимает большую красную кнопку «ограничение напряжения — аварийное переопределение». Вы теряете счёт, сколько раз ей нужно подтвердить «да, я уверена», и украшаете холл немного, пока ждёте.

Теперь вам нужно сделать наибольшее напряжение, включив **ровно двенадцать батарей** в каждом блоке.

Выходное напряжение блока всё ещё формируется из цифр включённых батарей; единственное отличие в том, что теперь в выходном напряжении будет **12 цифр** вместо двух.

Рассмотрим пример из предыдущей части:

```
987654321111111
811111111111119
234234234234278
818181911112111
```


Теперь напряжения значительно больше:

- В блоке `987654321111111` наибольшее напряжение получается, если включить все батареи, кроме нескольких единиц в конце, чтобы получить `987654321111`.
- В последовательности цифр `811111111111119` наибольшее напряжение получается, если включить все батареи, кроме нескольких единиц, чтобы получить `811111111119`.
- В блоке `234234234234278` наибольшее напряжение получается, если исключить батарею с цифрой 2, батарею с цифрой 3 и ещё одну батарею с цифрой 2 в начале, чтобы получить `434234234278`.
- В блоке `818181911112111` наибольшее напряжение получается, если исключить несколько единиц в начале, чтобы получить `888911112111`.

Суммарное выходное напряжение теперь значительно больше: 

`987654321111 + 811111111119 + 434234234278 + 888911112111 = 3121910778619`
