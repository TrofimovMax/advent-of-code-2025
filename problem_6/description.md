# День 6: Мусоросборник

После того как вы помогли эльфам на кухне, вы решили сделать паузу и помочь им инсценировать сцену из фильма. Однако, переполнившись энтузиазмом, вы слишком активно прыгнули в мусоропровод!

После короткого падения вы оказываетесь в дробилке для мусора. К сожалению, дверь была магнитно запаяна.

Когда вы пытаетесь найти выход, к вам подходит семья головоногих! Они уверены, что смогут открыть дверь, но это займёт некоторое время. Пока вы ждёте, они интересуются, можете ли вы помочь младшему головоногому с её [математическим заданием](https://adventofcode.com/2021/day/18).

Математика головоногих не сильно отличается от обычной. Рабочий лист (ваш входной файл) состоит из списка задач; каждая задача содержит группу чисел, которые нужно либо сложить (+), либо умножить (*) вместе.

Однако задачи расположены немного странно: они представлены рядом друг с другом в очень длинном горизонтальном списке. Например:

```
123 328  51 64 
 45 64  387 23 
  6 98  215 314
*   +   *   +  
```

Числа каждой задачи расположены вертикально; внизу задачи указан символ операции, которую нужно выполнить. Задачи разделены полной колонкой, содержащей только пробелы. Выравнивание чисел по левому/правому краю в задаче можно игнорировать.

В этом примере рабочий лист содержит четыре задачи:

```
123 * 45 * 6 = 33210
328 + 64 + 98 = 490
51 * 387 * 215 = 4243455
64 + 23 + 314 = 401
```

Чтобы проверить работу, студентам головоногих дают **итоговую сумму**, полученную сложением всех ответов по отдельным задачам. В этом рабочем листе итоговая сумма:

`33210 + 490 + 4243455 + 401 = 4277556`


Конечно, настоящий рабочий лист гораздо шире. Нужно убедиться, что вы корректно прочитали все задачи полностью.

**Задача:** решите все задачи на рабочем листе. Какова итоговая сумма, полученная сложением всех ответов отдельных задач?

## --- Часть Вторая ---
Большие головоногие возвращаются, чтобы проверить, как идут дела. Когда они видят, что ваш общий результат (`grand total`) не совпадает с ожидаемым по листу заданий, они понимают, что забыли объяснить, как читать «математику головоногих».

Математика головоногих записана справа налево по колонкам. Каждое число указано в своей колонке, при этом старший разряд находится вверху, а младший — внизу. (Задачи по-прежнему разделены колонкой, состоящей только из пробелов, а символ внизу задачи — это оператор, который нужно использовать.)

Пример листа заданий:

```
123 328  51 64
45 64  387 23
6 98  215 314
*   +   *   +
```

Если читать задачи справа налево по колонкам, задачи становятся совсем другими:

- Правая задача: 4 + 431 + 623 = 1058
- Вторая справа задача: 175 * 581 * 32 = 3253600
- Третья справа задача: 8 + 248 + 369 = 625
- Левая задача: 356 * 24 * 1 = 8544

Теперь общий результат: 1058 + 3253600 + 625 + 8544 = 3 263 827.

Решите задачи на листе ещё раз. Каков общий результат (`grand total`) после сложения всех ответов по отдельным задачам?
